## 🚀 Local Development

No build step is required. Clone the repo and open the file directly in a browser.

```bash
# 1. Clone the repository
git clone https://github.com/Asmaijaz/Portfolio.git

# 2. Navigate into the folder
cd Portfolio

# 3. Open in browser
open index.html
# or on Linux:
xdg-open index.html
# or on Windows:
start index.html
```

## 📦 Deployment (GitHub Pages)

This site is deployed via **GitHub Pages** directly from the `main` branch root.

To deploy your own fork:

1. Go to your repository **Settings → Pages**
2. Under **Source**, select `Deploy from a branch`
3. Choose **`main`** branch and **`/ (root)`** folder
4. Click **Save** — your site will be live at `https://<username>.github.io/<repo>/`
